# 로또 번호 추천 + 오늘의 운세

웹에서 동작하는 로또 번호 추천 서비스입니다.

- **로또 추천**: 1~45 중 6개 + 보너스 번호 랜덤 생성
- **오늘의 운세**: 100가지 운세 중 하나를 추첨통에서 뽑으면, 그에 맞는 고정 로또 번호 표시

## 실행 방법

브라우저에서 `index.html`을 열거나, 로컬 서버로 실행:

```bash
npx -y serve .
```

## GitHub 저장소

- https://github.com/Thinking6859/20260311vibe

### 처음 한 번 (저장소 연결)

Git이 설치된 상태에서:

```bash
git init
git remote add origin https://github.com/Thinking6859/20260311vibe.git
git add -A
git commit -m "Initial commit: 로또 추천 + 운세 추첨통"
git branch -M main
git push -u origin main
```

또는 `push-to-github.bat` 더블클릭 후 안내에 따라 진행하면 됩니다.

### 이후 업데이트 푸시

```bash
git add -A
git commit -m "업데이트 내용 요약"
git push
```
